# 🔐 Phishing Detection System

This project is a **Phishing URL Detection System** that checks whether a given website URL is **Safe or Phishing** using Machine Learning.
---
## 🚀 Features
- Detects phishing URLs in real-time
- Simple and user-friendly interface
- Machine Learning model (Random Forest)
- Fast prediction using Flask API
---

## 🛠️ Tech Stack
### Frontend:
- HTML
- CSS
- JavaScript

### Backend:
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
